# @joyful/turboq

## Installation

```bash
# npm
npm install @joyful-tools/turboq

# pnpm
pnpm add @joyful-tools/turboq

# bun
bun add @joyful-tools/turboq

# npm (from JSR)
npx jsr add @joyful/turboq

# pnpm 10.9+ (from JSR)
pnpm add jsr:@joyful/turboq

# yarn 4.9+ (from JSR)
yarn add jsr:@joyful/turboq

# deno
deno add jsr:@joyful/turboq
```
