# @joypack/waba

WhatsApp Business API tooling.

## Installation

```bash
# npm
npm install @joypack/waba

# pnpm
pnpm add @joypack/waba

# bun
bun add @joypack/waba

# npm (from JSR)
npx jsr add @joypack/waba

# pnpm 10.9+ (from JSR)
pnpm add jsr:@joypack/waba

# yarn 4.9+ (from JSR)
yarn add jsr:@joypack/waba

# deno
deno add jsr:@joypack/waba
```
